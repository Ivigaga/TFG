custom_widgets module

MÓDULO custom_widgets.py

Contiene componentes visuales (Widgets) personalizados creados desde cero mediante la API de dibujo de Qt (QPainter).

class custom_widgets.QRangeSlider(parent=None)[fuente]

Bases: QWidget

Control deslizante (Slider) personalizado con DOS selectores (thumbs). Permite definir un rango de zona muerta (valor mínimo y máximo) y muestra visualmente una barra de progreso que indica el estado del sensor actual.

adjust_high_thumb(delta)[fuente]

Ajusta programáticamente (mediante botones +/-) el marcador de límite superior.

adjust_low_thumb(delta)[fuente]

Ajusta programáticamente (mediante botones +/-) el marcador de límite inferior.

mouseMoveEvent(event)[fuente]

Permite arrastrar el selector activo actualizando su valor lógico.

mousePressEvent(event)[fuente]

Detecta sobre cuál de los dos selectores ha hecho clic el usuario basándose en proximidad.

paintEvent(event)[fuente]

Método central de renderizado de Qt. Dibuja geométricamente todas las partes del Slider. Se ejecuta cada vez que el widget necesita actualizarse visualmente.

set_current_value(val)[fuente]

Actualiza la barra azul de progreso en tiempo real basada en el movimiento del usuario.
